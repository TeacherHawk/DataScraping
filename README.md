Dette er et eksempel på hvordan man kan datascrape fra forskjellige nettsider for å bruke dataene derfra
Hittil så er det mulig å velge mellom:
- DNT: Liste over alle betjente hytter som DNT har og kan se hvilke områder de ligger i.
- FIS: Alle norske alpinister som har blitt registrert i FIS
- FRUKT: Mulighet å se en liste av forskjellige frukt, grønnsaker og bær
- MATPRAT: En samling av ulike matretter som ligger på matprat med hvilke rating, tidsbruk og vansklighetsgrad
